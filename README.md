# ☀️ SolarGuard AI App

Αυτή είναι μια ιδιωτική εφαρμογή Τεχνητής Νοημοσύνης για την αυτόματη ανίχνευση βλαβών σε φωτοβολταϊκά πάνελ (π.χ. ρωγμές, σκόνη, κτλ).

## 🛠️ Τεχνολογίες που χρησιμοποιήθηκαν:
* **Τεχνητή Νοημοσύνη:** YOLO (Ultralytics) - Εκπαιδευμένο μοντέλο (best2.pt)
* **Ιστοσελίδα / UI:** Streamlit
* **Επεξεργασία Εικόνας:** OpenCV & PIL

## 🚀 Πώς να το τρέξεις τοπικά (στο λάπτοπ):
1. Βεβαιώσου ότι έχεις εγκαταστήσει τα εργαλεία από το `requirements.txt`.
2. Άνοιξε το Terminal στον φάκελο του project.
3. Τρέξε την εντολή:
   ```bash
   streamlit run web.py
